# 项目简介

## 核心功能
- 电影数据采集
- 情感分析
- 数据存储与管理
- 可视化分析
- 协同过滤推荐

## 适用场景
- 电影爱好者分析电影趋势
- 电影网站个性化推荐
- 电影行业数据分析

## 技术特点
- Python语言实现高效数据处理
- Vue前端框架提供良好用户体验
- Flask后端框架稳定可靠
- 深度学习LSTM算法进行情感分析
- 双协同过滤推荐算法实现个性化推荐
- Echarts可视化工具辅助数据分析
- scrapy爬虫技术快速获取电影数据
- MySQL数据库保证数据存储安全
